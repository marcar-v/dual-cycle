# Dual Cycle

Aplicación para llevar un registro del ciclo menstrual de dos o más personas distintas. 

## Objetivo
- Encontrar patrones en el ciclo
- Compartir información del ciclo con otro usuario 
- Control de ciclo menstrual (duración, espacio entre ciclos)
- Previsión de ciclo 
- Registro cambios de humor

## Usuarios
- Personas menstruantes
- Personas que quieran acceder a los datos del ciclo de otro usuario

## Características de la aplicación
- Se tiene que poder consultar desde un móvil
- Autenticación de usuarios
- Acceso desde cualquier localización
- Permitir al usuario añadir y editar su ciclo
- Permitir al usuario añadir y editar los ciclos de otros usuarios
- Predecir el inicio y el fin del siguiente ciclo de un usuario

## Posibles funcionalidades
- Notificación de inicio del ciclo
- Recordatorio de introducción de ciclo
- Sincronización con otras apps
- Registro de fertilidad
- Seguimiento de embarazo
